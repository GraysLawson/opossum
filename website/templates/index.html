Welcome to OpossumBot

Your intelligent Discord companion for image analysis and server management

Features

  • Automatic image detection in Discord channels
  • Generate detailed descriptions of uploaded images
  • Customizable active channels
  • Easy configuration through web interface
  • Role assignment system with customizable buttons
  • OpenAI model selection for image analysis

How It Works

  1. Upload an image to a Discord channel
  2. OpossumBot detects the image
  3. Click the "Describe Image" button
  4. Receive a detailed description of the image
  5. Use role assignment buttons to manage server roles
  6. Configure bot settings through the web interface

New Features

Role Assignment

Easily manage server roles with customizable buttons. Users can assign or remove roles with a single click.

OpenAI Model Selection

Choose from various OpenAI models for image analysis, allowing you to balance between accuracy and performance.
